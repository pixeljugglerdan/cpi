A plugin to display a notification to conform with the EU Cookie Directive.

== Description ==

A plugin to display a notification to the user about the usage of cookies on the site. It allows the site admin to easily conform with the <a href='http://www.ico.gov.uk/news/latest_news/2011/must-try-harder-on-cookies-compliance-says-ico-13122011.aspx'>EU Cookie Directive</a>.

Developed by <a href="http://www.kinocreative.co.uk">Kino Creative</a>.
